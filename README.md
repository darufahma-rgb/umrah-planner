# El Massa Tour & Travel

Website resmi itinerary Umrah 2026 dari **El Massa Tour & Travel**.

## Tentang Proyek

Landing page yang menampilkan jadwal keberangkatan Umrah 2026, termasuk itinerary lengkap untuk bulan Juli dan Oktober.

## Teknologi

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Menjalankan Proyek

```sh
npm install
npm run dev
```

## Build Produksi

```sh
npm run build
```
